Hapus Akun

Hapus Toko Default

Menghapus toko default (termasuk pengaturan toko). Akun tetap ada.

Tips Keamanan

  • Kalau kamu ingin mengganti setting, gunakan halaman Pengaturan.
  • Hapus toko default dapat menyebabkan halaman pengaturan gagal memuat sampai toko default dibuat lagi.
